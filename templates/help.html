Help Page

Instructions for using this web app:

1.) Train an experiment simulator:

Here is the list of arguments that you must supply to train your experiment simulator:

batch_size = number of training points to be included in each epoch

reg = ???

hidden_act = hidden activation function you can choose from the following list

{}

out_act = output activation function you can choose from the following list

{}

max_epochs = number of training epochs

feature_transform = data transformation for features

target_transform = data transformation for targets

save = True or False whether or not you would like to save the trained experiment simulator

2.) Perform an optimization campaign:

Here is the list of arguments that you must supply to perform a cmapaign

algorithm = the optimization algorithm you would like to use (Phoenics)

goal = the optimization goal (maximize or mininmize)

max_iter = the number of suggested observations

3.) In each instance you can choose to recieve a file containing some data, given you provide your email.

If you would like to receive a file containing the training scores for that model please type "yes" in the appropriate field.

If you would like to receive a file containing the suggested observations the algorithm suggest type "yes" in the appropriate field.
